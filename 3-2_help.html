3-1학기 권장과목(택5)

1. 국어 연계과목
가. 화법과 작문 -> 심화국어
나. 언어와 매체 -> 문학과 매체

2. 영어 : 심화영어 독해Ⅰ

3. 수학
가. 인문/예체능 계열 : 수학적 사고와 통계
나. 자연 계열 : 수학과제탐구

4. 탐구
가. 인문계열 : 사회과목 2과목
(※연계과목: 한국지리->여행지리, 생활과 윤리->고전과 윤리, 사회문화->사회문제탐구)
나. 자연계열 : 3-1학기에 들었던 과학과목 중 실험과목 2개 선택

5. 예체능/정보계열 희망학생은 탐구과목 1개 대신 다음 과목 수강 가능
가. 음악 : 음악 연주
나. 미술 : 미술 감상과 비평
다. 체육 : 체육 전공 실기 응용
라. 정보 : 사물인터넷

※ 희망하는 대학 학과에서 권장하는 과목과 수능에서 원하는 과목을 고려하여 선택해주세요.
