2-2학기 권장과목(택3)

1. 인문계열 희망학생 : 수학Ⅱ + 사회 2과목 선택
(※ 한국지리, 사회문화, 생활과 윤리는 수능을 고려하여 3학년 1학기 수강 권장)

2. 자연계열 희망학생 : 미적분 + 과학 2과목 선택

3. 예체능/정보 계열 희망학생
가. 음악 : 수학Ⅱ + 턈구 1과목 +음악 전공 실기
나. 미술 : 수학Ⅱ + 턈구 1과목 +미술사
다. 체육 : 수학Ⅱ + 턈구 1과목 +체육 전공 실기 기초
라. 정보 : 미적분 + 턈구 1과목 +인공지능 기초

※ 희망하는 대학 학과에서 권장하는 과목과 수능에서 원하는 과목을 고려하여 선택해주세요.
