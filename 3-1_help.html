3-1학기 권장과목(택7)

1. 국어 : 화법과 작문 혹은 언어와 매체 중 택1

2. 영어 : 영어독해와 작문(일반) 혹은 심화 영어Ⅰ(진로) 중 택1

3. 수학
가. 인문/예체능 계열 : 확률과 통계
나. 자연 계열 : 통합수학Ⅰ + 1과목 선택(희망하는 경우)

4. 탐구
가. 인문계열 : 사회과목 2과목
(※연계과목: 세계지리->한국지리, 윤리와 사상->생활과 윤리, 사회탐구방법->사회문화)
나. 자연계열 : 2-2학기에 들었던 과학과목 중 Ⅱ과목 선택

5. 기타 원하는 과목 선택
가. 음악 : 음악사
나. 미술 : 미술 창작
다. 체육 : 체육 전공 실기 심화
라. 정보 : 정보 처리와 관리

※ 희망하는 대학 학과에서 권장하는 과목과 수능에서 원하는 과목을 고려하여 선택해주세요.
